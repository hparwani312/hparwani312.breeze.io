To enable your ICICIdirect account for using the Breeze API, you'll first need to have an active ICICIdirect account. Then, you'll need to create and register an app within the Breeze API platform, which involves logging in to the Breeze API portal using your ICICIdirect credentials. Finally, you'll generate a session key for your app, which allows you to interact with the Breeze API and use its features. 
Here's a more detailed breakdown:
  1. Have an ICICIdirect account:
    Ensure you have an active ICICIdirect trading account. 
  2. Access the Breeze API portal:
    Go to the Breeze API login page and log in using your ICICIdirect credentials. 
  3. Register an app:
    • In the Breeze API portal, you'll find an option to "Register App" or "View Apps". 
    • Enter a name for your app and any required redirect URL information. 
    • Submit the app registration. 
  4. Generate a session key:
    • Navigate to the "View Apps" tab in the Breeze API portal. 
    • Locate your registered app and click on "Login" or "Generate Session Key". 
    • You'll be redirected to a new webpage where you can log in with your ICICIdirect credentials and OTP. 
    • After successful login, your session key will be available either as part of the URL in the address bar or within the API Session option on the left side. 
  5. Use the session key:
    The generated session key is valid for 24 hours and allows you to use the Breeze API features. 
  6. ICICI Direct provides an SDK (Software Development Kit) called Breeze-Connect, which can simplify the process of using the Breeze API. You can install it using pip: pip install breeze-connect.
